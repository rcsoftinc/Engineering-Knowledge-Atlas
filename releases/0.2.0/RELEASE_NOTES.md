# Release v0.2.0 — Project Core

## Summary

This release establishes the permanent base of the Engineering Knowledge Atlas.

It defines the project's mission, identity, workflow, governance, knowledge model and operational separation.

## Key outcome

Future growth should primarily add new Knowledge Objects and update operational files without repeatedly rewriting the constitutional base.

## Breaking changes

The earlier experimental `project/` and `docs/` structure is replaced by `atlas/`, `operations/`, `templates/` and content directories such as `domains/`, `flows/` and `projects/`.

## Migration

Remove previous v0.2.0 draft folders before copying this release. Keep the existing Git history and replace only the working tree content.
