# Engineering Knowledge Atlas

> **Understand the system, not just the technology.**

Engineering Knowledge Atlas (EKA) is an open, versioned and AI-native knowledge system that maps the concepts, relationships, tools, practices and real-world flows that shape modern engineering.

The Atlas begins with understanding: what problem a concept solves, why it exists, where it belongs, what it depends on, what it enables, how it relates to other concepts, how it appears in real systems and how a person can reason about it with an AI.

## Mission

Organize engineering knowledge so people and artificial intelligence can learn, reason and build together.

## Repository map

- `atlas/` — constitutional and architectural core.
- `operations/` — current state, backlog, proposals and release checklists.
- `templates/` — schemas for future knowledge objects.
- `domains/` — knowledge domains and future concepts.
- `flows/` — end-to-end technical flows.
- `projects/` — real-world projects that connect many concepts.
- `comparisons/` — structured comparisons.
- `rabbit-holes/` — guided concept chains.
- `releases/` — release notes and version records.
